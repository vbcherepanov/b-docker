# Security Reports

Эта директория содержит отчеты сканирования безопасности.

## Генерация отчетов

Отчеты генерируются автоматически при запуске сканирования:

```bash
# Быстрое сканирование
./scripts/security-scan.sh quick

# Полное сканирование
./scripts/security-scan.sh full
```

## Типы отчетов

- `zap-*-report.html` - OWASP ZAP сканирование веб-уязвимостей
- `nikto-report.html` - Nikto сканирование веб-сервера
- `testssl-report.html` - TestSSL проверка SSL/TLS
- `trivy-report.json` - Trivy сканирование Docker образов
- `bitrix-security.txt` - Bitrix-специфичные проверки
- `summary-*.html` - Сводный отчет

## Просмотр отчетов

HTML отчеты можно открыть в любом браузере.

⚠️ **Важно**: Отчеты могут содержать чувствительную информацию о вашей системе.
Не публикуйте их в открытом доступе!
